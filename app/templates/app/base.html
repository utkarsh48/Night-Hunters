How To Use
Click-Jacking CheckSec has made this very easy for you to check whether your website is vulnerable to Click-Jacking attack or not. User just need to put URL of website in given search box. If the website opens within page without redirection, it means that website is vulnerable. The secured website will not get open and will show some error to that.
Port Scanner CheckSec will allow you to check for open ports within some range. You can change the range also in code according to your requirment. User needs to put IP address for which he/she wants to check it and it would be helpful in securing the network. Keep calm while using he tool because it take time for scan for ports and providing you accurate information.
Email Bombing CheckSec provides you the GUI interface to check whether the targeted email-id is prone to mass mailing or not. You need to fill the required inputs demanded for proper functioning of this tool. If the targeted email is secured then the response will be Your Email is secure otherwise the attack would be successful in mass mailing.
